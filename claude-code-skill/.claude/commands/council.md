# /council - Persona Council

Run a five-persona council (Contrarian, First-Principles Skeptic, Expansionist, Outsider, Executor) plus Chairman synthesis on a decision, draft, or proposal you're about to commit to.

Pass your input as the argument:
- `/council Should I accept the role at X?`
- `/council Review this draft: <paste>`
- `/council Architecture: should we use Postgres or DynamoDB for the new service`

Read and follow `skills/council/SKILL.md` for the full orchestration.
